# iime-skin
冰凌输入法（形码）皮肤，修改自用，非原创

## 2 款修改皮肤
- 微软五笔 UI 简洁优雅，冰凌五笔也内置了该风格皮肤。
- 基于该内置皮肤，我修改配色，并用 IrfanView 批量替换图片颜色，得到了 清华紫、北大红 两款皮肤。
- 未必适合长期使用，仅作纪念。
